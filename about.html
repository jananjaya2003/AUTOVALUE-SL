About This Project

A machine learning system for predicting used car prices in Sri Lanka, developed as a AI academic project.

Project Overview

AutoValue SL is a complete end-to-end machine learning project that predicts used car prices in Sri Lanka using real-world data scraped from online vehicle marketplaces. The system ingests 9,788 car listings, applies rigorous data preprocessing and feature engineering, trains and compares 11 regression models, and deploys the best model via a modern Flask web application.

The project demonstrates mastery of the full data science lifecycle — from raw data to production deployment — with special emphasis on detecting and mitigating model overfitting and underfitting.
